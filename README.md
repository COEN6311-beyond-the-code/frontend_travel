## Getting Started
First, make sure you have Node.js installed on your machine. You can download it from [here](https://nodejs.org/en/download/).

Then, make sure you're in the root directory of the project and run the following command to install the required dependencies:

```bash
npm install
```

Then, clone the repository to your local machine:


First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.tsx`. The page auto-updates as you edit the file.

## Building the project
To build the project, run the following command:

```bash
npm run build
```

## Running the production server
To run the production server, run the following command:

```bash
npm run start
```
